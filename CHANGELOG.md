# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-03-05

### Added
- Initial release.
- Visual **Conditional Logic** controls for CS Forms fields.
- Rule matching modes: **All (AND)** / **Any (OR)**.
- Operators: `==`, `!=`, `contains`, `not_contains`, `empty`, `not_empty`.
- Front-end field show/hide behavior.
- Hidden-field submit/validation handling.
- Support for builder preview and published pages.
